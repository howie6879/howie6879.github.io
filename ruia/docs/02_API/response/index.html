    Response #

    Response的目的是返回一个统一且友好的响应对象，主要属性如下：

    • url：请求的资源链接
    • metadata：跨请求传递的一些数据
    • html：源网站返回的资源数据
    • cookies：网站 cookies
    • history：访问历史
    • headers：请求头
    • status：请求状态码
