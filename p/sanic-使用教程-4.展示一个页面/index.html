Sanic дҪҝз”Ёж•ҷзЁӢ - 4.еұ•зӨәдёҖдёӘйЎөйқў

Jan 05, 2018

еүҚйқўдёҖиҠӮд»Ӣз»ҚйЎ№зӣ®з»“жһ„зҡ„ж—¶еҖҷпјҢеҫҲзІ—з•Ҙең°и®ІдәҶдёӢеҰӮдҪ•е°Ҷrssзҡ„ж–Үз« еҶ…е®№еңЁзҪ‘йЎөдёҠиҝӣиЎҢеұ•зӨәгҖӮ

зӣёдҝЎдҪ еә”иҜҘе·Із»ҸдәҶи§Јжё…жҘҡпјҢsanicжҳҜжҖҺд№ҲжҺҘж”¶иҜ·жұӮе№¶иҝ”еӣһиў«иҜ·жұӮзҡ„иө„жәҗзҡ„пјҢз®ҖеҚ•жқҘиҜҙжҰӮжӢ¬еҰӮдёӢпјҡ

  • жҺҘж”¶иҜ·жұӮ
  • жүҫеҲ°еҜ№еә”зҡ„и·Ҝз”ұе№¶жү§иЎҢи·Ҝз”ұеҜ№еә”зҡ„и§ҶеӣҫеҮҪж•°
  • Jinja2жЁЎжқҝжёІжҹ“иҝ”еӣһи§Ҷеӣҫ

и·Ҝз”ұе’Ңи§ҶеӣҫеҮҪж•°

еңЁжӯӨжҲ‘еҒҮи®ҫдҪ зҗҶи§Ј python дёӯзҡ„иЈ…йҘ°еҷЁпјҢеҰӮжһңдҪ е№¶дёҚжё…жҘҡпјҢеҸҜд»ҘзңӢжҲ‘еҸҰеҶҷзҡ„е…ідәҺиЈ…йҘ°еҷЁзҡ„д»Ӣз»ҚпјҢеӣһеҪ’жӯЈйўҳпјҢиҝҳи®°еҫ—з¬¬дёҖиҠӮдёӯзҡ„д»Јз Ғе®һдҫӢд№Ҳпјҹ

#!/usr/bin/env python
from sanic import Sanic
from sanic.response import text

app = Sanic()

# жӯӨеӨ„е°Ҷи·Ҝз”ұ / дёҺи§ҶеӣҫеҮҪж•° test е…іиҒ”иө·жқҘ
@app.route("/")
async def test(request):
    return text('Hello World!')


if __name__ == "__main__":
    app.run(host="0.0.0.0", port=8000)

еңЁеүҚиЁҖд»Ӣз»ҚйҮҢпјҢеҮәзҺ°иҝҷеҮ дёӘеҗҚиҜҚ и·Ҝз”ұ и§ҶеӣҫеҮҪж•° и§Ҷеӣҫ ,еңЁдёҠйқўйӮЈж®өд»Јз ҒдёӯпјҢtest е°ұжҳҜи§ҶеӣҫеҮҪж•°гҖӮ

иҝҷжҳҜдёҖж®өжү§иЎҢйҖ»иҫ‘пјҢжҜ”еҰӮе®ўжҲ·з«ҜиҜ·жұӮ 0.0.0.0:8000/ жӯӨж—¶иҝ”еӣһзҡ„еҶ…е®№е°ұжҳҜз”ұtest иҝҷдёӘи§ҶеӣҫеҮҪж•°жҸҗдҫӣзҡ„гҖӮ

еңЁжҲ‘зңӢжқҘпјҢи§ҶеӣҫеҮҪж•°е°ұжҳҜдёҖдёӘзәҪеёҰпјҢе®ғиө·еҲ°жүҝдёҠеҗҜдёӢзҡ„дҪңз”ЁпјҢйӮЈд№ҲпјҢеҲ°еә•жҳҜжҖҺж ·зҡ„жүҝдёҠеҗҜдёӢе‘ўпјҹи®©жҲ‘д»¬з»“еҗҲд»Јз Ғ(sanic0.1.2жәҗз Ғ)жқҘеҲҶжһҗдёӢпјҡ


@app.route("/")
async def test(request):
    return text('Hello World!')

иҝҷдёӘи·Ҝз”ұиЈ…йҘ°еҷЁзҡ„дҪңз”ЁеҫҲз®ҖеҚ•пјҢе°ұжҳҜе°Ҷ / иҝҷдёӘ uri дёҺи§ҶеӣҫеҮҪж•°testе…іиҒ”иө·жқҘпјҢжҲ–и®ёдҪ еҸҜд»Ҙе°Ҷи·Ҝз”ұжғіиұЎжҲҗдёҖдёӘ dictпјҢеҪ“е®ўжҲ·з«ҜиӢҘиҜ·жұӮ 0.0.0.0:8000/пјҢи·Ҝз”ұе°ұдјҡget / еҜ№еә”зҡ„и§ҶеӣҫеҮҪж•°testпјҢз„¶еҗҺжү§иЎҢгҖӮ

е…¶е®һзңҹе®һжғ…еҶөе’ҢжҲ‘д»¬жғіиұЎзҡ„е·®дёҚеӨҡпјҢиҜ·зңӢ sanic.py дёӯзҡ„з¬¬дёүеҚҒиЎҢ:


# Decorator
def route(self, uri, methods=None):

    def response(handler):
        # и·Ҝз”ұзұ»зҡ„add ж–№жі•е°Ҷи§ҶеӣҫеҮҪж•°handler дёҺuri е…іиҒ”иө·жқҘ
        # з„¶еҗҺж•ҙдёӘи·Ҝз”ұеҲ—иЎЁдјҡж–°еўһдёҖдёӘ namedtuple еҰӮдёӢпјҡ
        # Route(handler=handler, methods=methods_dict, pattern=pattern, parameters=parameters)
        self.router.add(uri=uri, methods=methods, handler=handler)
        return handler

    return response

жӯӨж—¶пјҢи·Ҝз”ұе°ұе’Ң uri еҜ№еә”зҡ„и§ҶеӣҫеҮҪж•°е…іиҒ”иө·жқҘдәҶпјҢиҝҷе°ұжҳҜжүҝдёҠпјҢи·Ҝз”ұе’Ңи§ҶеӣҫеҮҪж•°е°ұжҳҜиҝҷж ·еҜ№еә”зҡ„е…ізі»гҖӮ 103иЎҢжңүдёӘhandle_requestеҮҪж•°пјҡ


async def handle_request(self, request, response_callback):
    """
    Takes a request from the HTTP Server and returns a response object to be sent back
    The HTTP Server only expects a response object, so exception handling must be done here
    :param request: HTTP Request object
    :param response_callback: Response function to be called with the response as the only argument
    :return: Nothing
    """

еҪ“жңҚеҠЎеҷЁзӣ‘еҗ¬еҲ°жҹҗдёӘиҜ·жұӮзҡ„ж—¶еҖҷпјҢhandle_requestеҸҜд»ҘйҖҡиҝҮеҸӮж•°дёӯзҡ„request.url жқҘжүҫеҲ°и§ҶеӣҫеҮҪж•°е№¶дё”жү§иЎҢпјҢйҡҸеҚіз”ҹжҲҗи§Ҷеӣҫиҝ”еӣһпјҢиҝҷдҫҝжҳҜжүҖи°“зҡ„еҗҜдёӢгҖӮ

е…¶е®һжөҸи§ҲеҷЁжҳҫзӨәзҡ„еҶ…е®№е°ұжҳҜжҲ‘д»¬жүҖи°“зҡ„и§ҶеӣҫпјҢи§ҶеӣҫжҳҜи§ҶеӣҫеҮҪж•°з”ҹжҲҗзҡ„пјҢе…¶дёӯJinja2иө·еҲ°жЁЎжқҝжёІжҹ“зҡ„дҪңз”ЁгҖӮ

и“қеӣҫ

еҲ°иҝҷйҮҢпјҢдҪ дёҖе®ҡе·Із»ҸеҫҲжҳҺзҷҪsanicжЎҶжһ¶жҳҜжҖҺд№ҲеӨ„зҗҶдёҖдёӘиҜ·жұӮе№¶е°Ҷи§Ҷеӣҫиҝ”еӣһз»ҷе®ўжҲ·з«ҜпјҢз„¶еҗҺд№ҹжҺҢжҸЎдәҶеҰӮдҪ•зј–еҶҷиҮӘе·ұе®ҡд№үзҡ„жЁЎжқҝ(html)д»ҘеҸҠж ·ејҸ(css)пјҢйҖҡиҝҮеүҚйқўдёҖиҠӮйЎ№зӣ®з»“жһ„зҡ„д»Ӣз»ҚпјҢжҲ‘д»¬еҸҜд»ҘжҖ»з»“еҮәеҰӮдёӢз»ҸйӘҢпјҡ

  • еҜ№дәҺcssгҖҒjsзӯүйқҷжҖҒж–Үд»¶пјҢеёёи§„ж“ҚдҪңжҳҜе°Ҷе…¶ж”ҫеңЁиҮӘе·ұе»әз«Ӣзҡ„staticsдёӢйқў
  • еҜ№дәҺhtmlжЁЎжқҝпјҢеёёи§„ж“ҚдҪңжҳҜе°Ҷе…¶ж”ҫеңЁиҮӘе·ұе»әз«Ӣзҡ„templatesдёӢйқў
  • и§ҶеӣҫеҮҪж•°(еҚіжңҚеҠЎзҡ„йҖ»иҫ‘е®һзҺ°)ж”ҫеңЁиҮӘе·ұе»әз«Ӣзҡ„viewsдёӢйқў

жҳҜж—¶еҖҷиҖғиҷ‘д»ҘдёӢиҝҷз§Қжғ…еҶөдәҶпјҢдҪ йңҖиҰҒзј–еҶҷдёҖдёӘжҜ”иҫғеӨҚжқӮзҡ„httpжңҚеҠЎпјҢдҪ йңҖиҰҒе®ҡд№үеҮ зҷҫдёӘи·Ҝз”ұпјҢеңЁзј–еҶҷиҝҮзЁӢдёӯдҪ дјҡеҸ‘зҺ°жңүи®ёеӨҡдёҚйЎәеҝғзҡ„ең°ж–№пјҡ

  • еҗ„з§ҚдёҚеҗҢзұ»еһӢзҡ„и·Ҝз”ұдә’зӣёдәӨжқӮеңЁдёҖиө·пјҢе‘ҪеҗҚеӣ°йҡҫпјҢз®ЎзҗҶеӣ°йҡҫ
  • дёҚеҗҢйЎөйқўзҡ„cssж–Үд»¶еҗҢж ·е Ҷз§ҜеңЁдёҖиө·пјҢhtmlд№ҹжҳҜеҰӮжӯӨ
  • …

е®һеңЁжҳҜд»ӨдәәзғҰжҒјпјҢеҸҜиғҪдҪ жғіиҰҒдёҖдёӘжЁЎеқ—еҢ–зҡ„зј–еҶҷж–№ејҸпјҢдёҖдёӘж–Үд»¶зј–еҶҷеҗҺеҸ°пјҢurlз»ҹдёҖжҳҜ/admin/***пјҢдёҖдёӘж–Үд»¶зј–еҶҷеҸ‘её–пјҢ/post/*** зӯүзӯүпјҢеҗ„дёӘж–Үд»¶дёӢйқўurlиҮӘеҠЁдјҡеёҰдёҠиҮӘе®ҡд№үзҡ„еүҚзјҖпјҢдёҚз”ЁиҖғиҷ‘е‘ҪеҗҚй—®йўҳпјҢдёҚз”ЁйҮҚеӨҚеҶҷurlеүҚзјҖпјҢеӨҡд№ҲзҫҺеҘҪ

BlueprintпјҢе°ұжҳҜsanicдёәдҪ жҸҗдҫӣзҡ„и§ЈеҶіж–№жЎҲпјҢдҫқз„¶жҳҜдёҠиҠӮrssзҡ„дҫӢеӯҗпјҢи®©жҲ‘д»¬еҲ©з”ЁBlueprintжқҘз®ҖеҚ•е®һзҺ°дёҖдёӢжҲ‘д»¬зҡ„йңҖжұӮпјҢжҜ”еҰӮжҲ‘д»¬иҰҒжһ„е»әзҡ„зҪ‘з«ҷеҲҶдёәдёӨдёӘйғЁеҲҶпјҡ

  • /json/index иҝ”еӣһjsonж јејҸзҡ„ж•°жҚ®
  • /html/index иҝ”еӣһhtmlи§Ҷеӣҫ

и“қеӣҫзҡ„зӣ®зҡ„е°ұжҳҜи®©жҲ‘д»¬жһ„е»әзҡ„йЎ№зӣ®зҒөжҙ»пјҢеҸҜжү©еұ•пјҢе®№жҳ“йҳ…иҜ»пјҢж–№дҫҝз®ЎзҗҶпјҢдёҖдёӘдёӘе°Ҹзҡ„и“қеӣҫе°ұжһ„е»әдәҶдёҖдёӘеӨ§еһӢзҡ„йЎ№зӣ®пјҢrun.pyйҮҢйқўеҸҜд»ҘйҡҸж„Ҹз»„еҗҲжіЁеҶҢи“қеӣҫпјҢеҸҜжҠҪжҸ’ејҸзҡ„жһ„е»әжҲ‘д»¬зҡ„жңҚеҠЎпјҢsample01йҮҢжҳҜжң¬ж¬ЎзӨәдҫӢзҡ„д»Јз ҒпјҢе»әи®®е…ҲиҜ»дёҖйҒҚпјҢеҚҒеҲҶз®ҖеҚ•пјҢдё»иҰҒе°ұжҳҜи®Іи·Ҝз”ұж №жҚ®дҪ иҮӘе·ұе®ҡд№үзҡ„зү№жҖ§еҲҶеүІжҲҗдёҖдёӘдёӘи“қеӣҫпјҢжҜ”еҰӮиҝҷйҮҢзҡ„rss_htmlе’Ңrss_json:


#!/usr/bin/env python
# йғЁеҲҶд»Јз Ғ
# rss_html.py
import sys

from sanic import Blueprint
from sanic.response import html

from src.config import CONFIG


html_bp = Blueprint('rss_html', url_prefix='html')
html_bp.static('/statics/rss_html', CONFIG.BASE_DIR + '/statics/rss_html')

@html_bp.route("/")
async def index(request):
    return await template('index.html')
